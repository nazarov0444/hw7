Флекс-контейнер

Flex-direction

  • row
  • row-reverse
  • column
  • column-reverse
  • row слева направо
  • row-reverse cправа налево
  • column cверху вниз
  • column-reverse снизу вверх

    • Flex-wrap

    • wrap
    • nowrap
    • wrap-reverse
    • nowrap (по умолчанию): все flex элементы будут в одной строке
    • wrap: flex-элементы будут перенесены на несколько строк сверху вниз.
    • wrap-reverse: flex-элементы будут перенесены на несколько строк снизу вверх.

    Justify-content

    • flex-start
    • flex-end
    • center
    • space-between
    • space-around
    • space-evenly
    • flex-start: элементы сдвинуты в начало
    • flex-end: элементы сдвинуты ближе к концу
    • center: элементы центрированы вдоль линии
    • space-between: элементы равномерно распределены по линии
    • space-around: элементы равномерно распределены по линии с одинаковым пространством вокруг них.
    • space-evenly: элементы распределяются таким образом, чтобы расстояние между любыми двумя элементами (и расстояние до краев) было одинаковым.

    Align-items

    • stretch
    • flex-start/start/self-start
    • flex-end/end/self-end
    • center
    • baseline
    • stretch: растягивать, чтобы заполнить контейнер
    • flex-start / start / self-start: элементы размещаются в начале поперечной оси.
    • flex-end / end / self-end: элементы располагаются в конце поперечной оси.
    • center: элементы центрированы по поперечной оси
    • baseline: элементы выровнены, по их базовой линии

    Align-content

    • flex-start/start
    • flex-end/end
    • center
    • space-between
    • space-around
    • space-evenly
    • stretch
    • flex-start / start: элементы, сдвинуты в начало контейнера.
    • flex-end / end: элементы, сдвинуты в конец контейнера.
    • center: элементы выровнены по центру в контейнере
    • space-between: элементы равномерно распределены; первая строка находится в начале контейнера, а последняя — в конце
    • space-around: элементы равномерно распределены с равным пространством вокруг каждой строки
    • space-evenly: элементы распределены равномерно, вокруг них одинаковое пространство
    • stretch: линии растягиваются, чтобы занять оставшееся пространство

    Order

    • order
    • order: управляет порядком их появления в контейнере flex.

    Flex-grow

    • flex-grow
    • flex-grow: установлено значение 1, оставшееся пространство в контейнере будет равномерно распределено между всеми дочерними элементами.

    Flex-shrink

    • flex-shrink
    • flex-shrink: Это свойство определяет способность гибкого элемента сжиматься при необходимости.

    Flex-basis

    • flex-basis
    • flex-basis: Это свойство определяет размер элемента по умолчанию перед распределением оставшегося пространства.

    Align-self

    • align-self
    • align-self: Это свойство позволяет переопределить выравнивание по умолчанию для отдельных элементов flex.
